## Integración con TMDB

Esta aplicación utiliza la API de [The Movie Database (TMDB)](https://developer.themoviedb.org/) para mostrar películas populares en tiempo real. El uso es estrictamente académico y no comercial.

### API Key
La clave se gestiona de forma local y no se expone públicamente. Se utiliza únicamente para fines educativos.

### Funcionalidad
- Consulta de películas populares
- Visualización de cartelera con imágenes y popularidad
